# towerclimber

https://aratama.github.io/platformer

### Guides

- Left and right button to move
- Up buton to look up
- Down button to crouch down or climb down a ladder
- Button1 to jump

### Run locally

```bash
$ w4 watch
```

### Release Build

Install wasm-snip:

```
$ cargo install wasm-snip
```

And:

```bash
$ ./build
```
